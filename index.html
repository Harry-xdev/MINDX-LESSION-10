Bài 1

person = {
name: 'Bob',
occupation: 'web developer',
hobbies: 'painting',
};

1/ Cách truy cập bằng dấu "."

  • person.name
  • person.occupation
  • person.hobbies

2/ Cách truy cập bằng dấu "[]"

  • person.["name"]
  • person.["occupation"]
  • person.["hobbies"]

Bài 4: Thay giá trị của 6 = 606

let obj = {
foo: { a: "hello", b: "world" },
bar: ["example", "mem", null, { xyz: 6 }, 88],
qux: [4, 8, 12]
};

1/ Cách truy cập trực tiếp

  • obj.bar[3].xyz = 606

2/ Cách truy cập bằng Object.assign

  • Object.assign(obj.bar[3],{xyz:606})
